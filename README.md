# Curso Spring Boot: Construindo uma API REST do Zero 🚀

Repositório dedicado aos códigos e anotações do curso de Spring Boot ministrado pelo canal **Souza DEV**. O objetivo principal deste projeto é aprender na prática como construir uma API REST robusta utilizando Java e o ecossistema Spring.

## 📚 Sobre o Projeto

Este projeto serve como uma base de estudos para o desenvolvimento de aplicações backend modernas. Em vez de focar em um único domínio, o foco está na evolução da arquitetura e no domínio das ferramentas essenciais para um desenvolvedor Java, abordando os seguintes pilares:

* **Desenvolvimento de APIs REST:** Criação de endpoints, manipulação de métodos HTTP (GET, POST, PUT, DELETE), tratamento de exceções e padronização de respostas JSON.
* **Persistência de Dados com JPA:** Implementação do Spring Data JPA para mapeamento objeto-relacional, configuração de repositórios e integração eficiente com bancos de dados.
* **Segurança com Spring Security:** Proteção de rotas, configuração de filtros de segurança, autenticação e controle de permissões de acesso aos recursos da API.

## 🛠️ Tecnologias e Ferramentas Utilizadas

* **Java** (Framework Spring Boot)
* **Spring Web**
* **Spring Data JPA**
* **Spring Security**
* **Maven** (Gerenciamento de dependências)
* **Postman** (Validação e testes de integração)
